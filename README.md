# Knowledge Asset Search
A Python project for searching and filtering knowledge assets.

## Usage
1. Create a list of `KnowledgeAsset` objects.
2. Initialize a `KnowledgeAssetSearch` object with the list of assets.
3. Use the `filter_by_category`, `filter_by_tags`, `sort_by_relevance`, `sort_by_last_updated`, or `sort_by_popularity` methods to search and filter the assets.

## Testing
Run the tests using `pytest` to ensure the implementation is correct.
